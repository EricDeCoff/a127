a127
====

Apigee 127 Command Line Interface


Usage
-----

  Usage: a127 [options] [command]

  Commands:

    account <action>
       manage deployment accounts
    
    project <action>
       manage a project
    
    usergrid <action>
       manage Usergrid process
    
    config 
       print config
    
    help [cmd]
       display help for [cmd]
    

  Options:

    -h, --help  output usage information
